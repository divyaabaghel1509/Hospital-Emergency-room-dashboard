# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Hospital Emergency Room data to provide 
real-time insights on patient flow, wait times, admission status, and 
department-wise referrals.

## 📊 Key Metrics
- **Total Patients:** 506
- **Average Patient Satisfaction Score:** 5.33
- **Average Patient Wait Time:** 35.88 mins
- **Admission Rate:** 50% Admitted | 50% Not Admitted

## 🔍 Dashboard Features
- **Age Group wise Patient Distribution**
- **Gender wise Analysis** (Male vs Female)
- **Admission Status** (Admitted / Not Admitted)
- **Wait Time Analysis** (On Time vs Delayed)
- **Department Referral Breakdown** (Orthopedics, Cardiology, Neurology, etc.)
- **Monthly Filter** (Jan to Dec)

## 🛠️ Tools Used
- Microsoft Excel (Dashboard & Data Visualization)
- Excel (Pivot Tables, Power Query, Charts)
- Excel Formulas (IF, COUNTIF, SUMIF, XLOOKUP)

## 💡 Key Insights
- 60-70 age group has the highest patient visits (74 patients)
- 40-50 age group has the lowest visits (45 patients)
- 312 patients were attended On Time vs 194 Delayed
- General Practice and Orthopedics are top referred departments
- Gender ratio is nearly equal — 275 Female, 231 Male
